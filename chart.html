📈 차트 엔진(TradingView 고급차트)을 아직 설치하지 않았어요

이 페이지는 무료 TradingView Advanced Charts(Charting Library)를 사용합니다. 라이브러리 파일은 TradingView에서 직접 받아 사이트에 올려야 해요(재배포 불가). 데이터 연동·테마·검색은 이미 준비돼 있어요.
  1. tradingview.com/advanced-charts 에서 무료 사용 신청(간단한 폼 → GitHub 저장소 접근 권한 부여)
  2. 받은 저장소의 charting_library/ 폴더를 이 사이트 루트(= index.html과 같은 위치)에 그대로 업로드
  3. 새로고침하면 이 자리에 차트가 뜹니다. (파일 경로는 charting_library/charting_library.standalone.js 여야 해요)
※ 데이터피드(datafeed.js)와 ohlc/*.json은 수집기가 생성합니다. 워크플로에서 git add ohlc가 포함되는지 확인하세요.
